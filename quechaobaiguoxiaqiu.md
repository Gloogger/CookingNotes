# 雀巢白果虾球
准备材料清单：
1. 意大利面
2. 虾
3. 葱蒜姜
4. 胡萝卜
5. 黄瓜

- 雀巢的制作
    - 意大利面
        1. 煮意大利面，煮至可以用手掐断即可捞出，不要过凉水
        2. 捞出后拍玉米淀粉，多拍一点
        3. 放入有孔模具中摆放成型
        4. 成型后将第二个模具按压在第一个模具上，如下图所示：
