# 炸大虾
## 准备材料清单：
1. 黄油
2. 白兰地
3. 鸡蛋
4. 面粉
5. 面包糠
6. 虾

- 虾的准备
    1. 虾去身皮、去虾线、去虾包，不去虾尾
    2. 在虾上加入一点料酒、盐、鸡精、摔打
    3. 把虾开边
    4. 拍粉拖蛋上面包糠
    5. 用菜刀拍实，整形
    
- 虾的烹制
    1. [炸]的油量，3-4成油温
    2. 加入开边虾
    3. 浮起后反面炸
    4. 两面炸过后捞出
    5. 洗锅
    6. 加入黄油，融化后加入适量白兰地（用手堵住瓶口淋一些即可）。出味后淋在虾上。
